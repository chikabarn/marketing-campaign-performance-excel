# Marketing Campaign Performance Analysis (Excel)

## Overview
This project analyzes digital marketing campaign performance across multiple channels using Excel. The goal is to identify high-performing channels and campaign types based on engagement and conversion metrics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Dataset
Digital marketing campaign dataset sourced from Kaggle.

## Key Metrics Analyzed
- Click-Through Rate (CTR)
- Conversion Rate
- Total Conversions
- Ad Spend

## Analysis Process
1. Data cleaning and validation
2. Pivot table analysis by channel and campaign type
3. Visualization of key performance metrics
4. Insight generation and recommendations

## Key Insights
- PPC and Social Media showed higher engagement rates.
- Email campaigns demonstrated higher conversion efficiency.
- Referral and PPC channels generated the highest total conversions.
- Higher ad spend did not always result in better performance.

## Recommendations
- Allocate budget based on conversion efficiency rather than engagement alone.
- Optimize high-traffic channels with lower conversion rates.
- Align campaign objectives with channel strengths.
